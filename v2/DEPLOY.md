# 即梦 Demo - 局域网部署指南

## 🚀 快速启动

### 1. 启动本地服务器

在项目目录执行：
```bash
cd /Users/bytedance/游戏化回流-v2
python3 -m http.server 5500
```

### 2. 获取局域网 IP

```bash
ifconfig | grep "inet " | grep -v 127.0.0.1 | head -1 | awk '{print $2}'
```

预期输出：`10.82.0.55`（或其他局域网 IP）

### 3. 在手机浏览器打开

确保手机和电脑在同一 WiFi 网络下，访问：

```
http://10.82.0.55:5500/
```

## 📱 可用页面

| 页面 | URL | 功能 |
|------|-----|------|
| 主 Demo | `http://10.82.0.55:5500/index.html` | 完整的即梦互动 demo |
| 算色预览 | `http://10.82.0.55:5500/color-preview.html` | 上传图片实时预览色调和颜色效果 |
| 背景参数调试 | `http://10.82.0.55:5500/bg-tuner.html` | 调整背景模糊、亮度、遮罩等参数 |

## 🛑 停止服务

按 `Ctrl+C` 停止服务器。

1. 编辑 `index.html`、`styles.css` 或其他文件

4. 如需清除缓存，可增加 `?v=YYYYMMDD-N` 版本号
